# Sergio Vision

Private presentation site for Sergio Djurdjevic (Sergios Skadecenter ApS, Ølstykke, Denmark). Built by MakerToo. Serbian-first (SR/DA toggle), same visual scaffold as Amstelbay — dark Executive Noir palette, GSAP animations, 7 sections.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

Next.js 16 | Tailwind v4 | GSAP 3 | TypeScript | React 19

## Build & Deploy

```bash
npm run build
```

Deployed via nixpacks on self-hosted infrastructure (Coolify).

## Structure

- `src/components/` — 7 animated section components + navbar + i18n provider
- `src/locales/` — SR/DA translation dictionaries
- `sergio-vision.md` — Project blurb
